Post-a-movie

This webapplication was made for the Client-side programming individual course.

This webapplication was made to showcase the different skills required to pass the course. It will contain different CRUD functionalities as wel as provide a comfortable user experience.

The webapplication provides the user with the ability to add and review movies. The library of movies is fully public, which means anyone can add or edit any movie, you could say it is like a wikipedia for movies. The user can also create lists of movies, like a ones to watch list. And share these lists with their friends.

Functionalities

These are the different things the application can do. Some might not yet be integrated at the time of reading.

Users

Full CRUD functionality for users (authentication needed). Be able to create, read, update and delete accounts.

Must

Movies

Full CRUD functionality for all users. Be able to add any movee, look them up, update a movie and delete them.

Must

Directors

Full CRUD functionality. Be able to create, read, update or delete any director and add them to a movie page.

Should

Ratings

Full CRUD functionality (with authentication). Be able to create, read, update or delete a rating/review.

Must

Lists

Full CRUD functionality (with authentication). Be able to create, read, update or delete a list.

Friends

Add other users to your friend list. When both users have added eachother they can share any personal list.
